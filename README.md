# Power BI Sales Dashboard

## Overview
An interactive Power BI dashboard designed to analyze sales performance across countries, products, and sales representatives.

## Features
- KPI Cards for Total Sales and Total Packages
- Sales Analysis by Country
- Product-wise Performance Tracking
- Interactive Slicers and Filters
- Geographic Sales Visualization
- Gauge Chart for Target Tracking

## Tools Used
- Power BI
- DAX
- Excel

## Key Insights
- Identified top-performing countries and products.
- Monitored sales achievement against targets.
- Analyzed package distribution across regions.
- Enabled interactive business reporting for decision-making.
